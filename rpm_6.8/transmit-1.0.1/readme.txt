CentOS6.8版本
==========================================
rpm -ivh json-c-0.10-1.x86_64.rpm

安装编译环境
==========================================
yum -y install gcc gcc-c++ autoconf automake

安装编译环境
==========================================
1. 带调试信息的编译：
   g++ -g transmit.c -o transmit -ljson
2. 不带调试信息的编译：
   g++ transmit.c -o transmit -ljson
3. 后台运行方式：
   /root/transmit/transmit &
4. 内存泄露检测工具：
   valgrind --tool=memcheck --leak-check=full --show-reachable=yes ./transmit
